# GomisuteWasurenai

GomisuteWasurenaiは鳴海地区限定のゴミの分別アプリです。

以下の困り事を解決する為に開発しました。
・何曜日が何ゴミを捨てるのかいまだに覚えられないので、スマホですぐに確認出来るようにしたい。
・不燃ゴミの収集日が月1回しかなく、出し忘れてしまいゴミが溜まってしまう。
・自宅の収納棚に貼ってある、分別マップをデジタル化したい。
